安装

Install

一条命令，永久生效。

One command, always on.

macOS / Linux
curl -fsSL https://raw.githubusercontent.com/CtosOyama/claude-router/main/install-remote.sh | bash
Windows PowerShell
git clone https://github.com/CtosOyama/claude-router.git $env:TEMP\skill-router; cd $env:TEMP\skill-router; powershell -ExecutionPolicy Bypass -File install.ps1

安装后重启 Claude Code 即可。卸载：rm -rf ~/.claude/skills/skill-router 并移除 hook 配置。

Restart Claude Code after install. Uninstall: rm -rf ~/.claude/skills/skill-router and remove the hook entry.

核心原理

How It Works

8 步调度算法，从收到消息到调用技能。

An 8-step algorithm, from receiving a message to invoking the right skill.

🚨

Step -1

紧急检测：如果消息包含"救命""立刻""崩了"等关键词，跳过所有步骤，立刻调度。

Emergency Check: Keywords like "urgent", "now", "broken", "down" skip all steps and route immediately.

🔍

Step 0.5

环境扫描：看一眼你的桌面、当前目录、历史记录，理解上下文。

Context Scan: Quick look at your desktop, CWD, and history to understand the situation.

🎯

Step 1-2

意图解析 + 信心打分：从你的话里提取"要做什么 + 在哪个领域 + 多大范围"，算出一个匹配分数。

Intent Parse + Score: Extract action + domain + scope, compute a weighted confidence score.

🔗

Step 3-4

匹配 + 编排：在 53 个技能中找到最佳匹配，决定是单走、串联还是并行。

Match + Orchestrate: Find the best skill among 53, decide solo / chain / parallel.

🛟

Step 5-7

兜底机制：信心不足时提出 3 个选项让你选，都不行就模糊匹配，再不行就用通用能力帮你。

Fallback: Presents 3 options when unsure. Then fuzzy match. Then generic handling. Never leaves you hanging.

📚

Step 8

自学习：追踪每次调度的反馈，对的加分错的扣分，越用越懂你。

Self-Learning: Tracks feedback on every route. Correct → boost. Wrong → demote. Gets better over time.
